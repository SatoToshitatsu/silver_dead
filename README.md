# HTML Game Project

このリポジトリは、HTML / JavaScript を用いて制作したWebゲームです。  
Three.jsなどの外部ライブラリを利用し、ブラウザ上で動作する3Dゲームの制作を目的としています。

## 概要

本作品では、HTML、CSS、JavaScriptを用いて、ブラウザ上で遊べるゲームを制作しています。  
GitHub Pagesを利用することで、URLを共有するだけで他の人もゲームをプレイできます。

## 使用技術

- HTML
- CSS
- JavaScript
- Three.js
- GitHub Pages
